# README

Demonstration: https://blog-app-greyknight.fly.dev

Rails Blog
A fully functional blogging platform built with Ruby on Rails. Users can create, edit, and delete posts, with intuitive management of content. Designed with maintainability in mind, it demonstrates clean Rails architecture, RESTful routing, and full CRUD functionality. Ideal as a foundation for extending features like authentication, commenting, or notifications.
Tech Stack: Ruby | Rails | SQLite/PostgreSQL | HTML | CSS
